# Onlybees Ticket Selection UI (Clone)

This project is a 1:1 clone of the Onlybees event ticket selection page built using **Next.js (React)**.

## Features
- Fetches live event section data from API
- Displays ticket sections with price
- Increase / decrease ticket quantity
- Bottom bar with total quantity & total price
- Proceed button logs final selection in console

## Tech Stack
- Next.js (App Router)
- React Hooks (useState, useEffect)
- JavaScript
- CSS

## API Used
https://concertsapi.onlybees.in/api/sections/availability
